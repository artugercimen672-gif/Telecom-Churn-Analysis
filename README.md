# 📊 Telecom Churn Analysis Project

This project uses **SQL Server** for data mining and **Python** for statistical visualization to identify why customers are leaving.

### 🚀 Key Business Insights (Öne Çıkan Bulgular)
- **Critical Threshold:** Churn rate jumps from **14% to 48%** after the 4th customer service call.
- **High-Risk Segment:** International plan users have a **43.7%** churn rate, which is 4x higher than others.
- **Cost Impact:** Churned customers have a higher average day charge (**31.37**) compared to loyal ones (**26.98**).

### 🛠️ Technical Skills (Teknik Yetkinlikler)
- **SQL Server:** Advanced querying, churn rate calculations, and regional analysis.
- **Python:** Data visualization using **Seaborn** and **Matplotlib** (Boxplots, Violin plots, Heatmaps).
- **Statistics:** Correlation analysis and variance examination.
